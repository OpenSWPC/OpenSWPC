# ディレクトリ構成

```
      ├── doc            : ドキュメント
      ├── bin            : 実行バイナリ
      ├── dataset        : データセットおよびデータ生成スクリプト
      ├── example        : 入力ファイルサンプル
      └── src            : ソースコード
          ├── swpc_3d    : 3次元問題
          ├── swpc_psv   : 2次元P-SV問題
          ├── swpc_sh    : 2次元SH問題 
          ├── tools      : 可視化等ツール群
          └── shared     : 共通モジュール
```
